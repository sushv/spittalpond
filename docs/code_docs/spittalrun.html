auto_create_random_numbers(random_number_table_name='rand_nums', number_of_chunks=10, number_of_rows_per_chunk=1000, number_of_pages=10, number_of_samples_per_page=20)[source]¶

Generate random numbers with the default Random Number Table Version.

Much of the time we just want to run Oasis without worry about these “random” (computers can only generate pseudo-random numbers anyways) numbers.

Use this method to quickly generate them for ease of use.

Note to Users: At the time of R1.3 there is, by default, a random number table version with the ID of 2. This can be used to avoid having to upload a random number table and create the version manually.
